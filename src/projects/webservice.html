Webservice rozszerzający możliwości Serwera Identyfikacji

Java 11 + SpringBoot
Kwiecień 2020 - Maj 2020

Wykorzystywany w pewnym projekcie darmowy serwer identyfikacji nie posiadał funkcji czasowego przydzielania uprawnień użytkownikom (delegacje). Serwer udostępniał REST API do przydzielania i odbierania uprawnień oraz wszystkich innych operacji na użytkownikach i rolach. Powstała aplikacja GUI do obsługi serwera z dodatkowymi funkcjami definiowania delegacji do którego trzeba było napisać webservice.

Aplikacja składała się z kontrolera udostępniającego metody do zarządzania delegacjami i modułu wykonującego delegacje za pomocą REST API udostępnionego przez Serwera Identyfikacji. Uwierzytelnienie zapytań do kontrolera było realizowane poprzez wywołanie uwierzytelnienia na Serwera Identyfikacji, dzięki czemu z punktu widzenia GUI zapytania do obu serwerów były jednolicie autoryzowane. Moduł wykonujący delegacje cyklicznie co stały interwał sprawdzał delegacje i w zależności od statusu dodawał lub odbierał uprawnienia, autoryzując się za pomocą specjalnie utworzonego konta technicznego.
